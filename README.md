# SDGG Card Viewer (SD건담외전 카드다스 뷰어)

정적 HTML 카드 뷰어입니다. `index.html` 하나와 이미지 폴더로 구성됩니다.

## 구성
- `index.html` — 뷰어 (카드 데이터 내장). 이미지는 상대경로 `.jpg`로 참조합니다.
- `1_SD건담외전/`, `2_신SD건담외전/`, `3_신약SD건담외전/`, `4_SD건담외전NEO/` — 카드 이미지(앞면) 폴더.

## 이미지 변환 (수동)
이미지는 현재 `.png`로 들어 있습니다. 뷰어는 `.jpg`를 참조하므로, 각 이미지를 **같은 경로/같은 파일명**으로 `.jpg`로 변환해 주세요. (크기 그대로, 권장 품질 q92~100)
변환 후 원본 `.png`는 삭제해도 됩니다.

## 배포 (GitHub Pages)
1. 이 폴더를 저장소에 push
2. Settings → Pages → Branch: main / root 선택
3. 공개 URL의 `index.html`로 접속

## 기능
- 제품 → 편 → 시리즈 탭 탐색
- 카드번호·이름·설명·수치·대사, 한글/일본어 토글
- 전체 시리즈 검색(기본: 이름·번호 / 체크박스: 대사·설명 포함)
- 행 클릭 시 우측에 원본 카드 이미지 표시
